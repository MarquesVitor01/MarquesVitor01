## 🏆 Projetos Recentes

### 🛒 1. Data Lakehouse E-commerce: Pipeline Medallion com PySpark & Power BI
*Construção de uma infraestrutura robusta para processamento distribuído de Big Data utilizando dados reais de e-commerce.*
* **Engenharia de Plataforma:** Ingestão de dados brutos salvos diretamente em **Databricks Volumes** (Camada Bronze).
* **Processamento de Dados:** Pipeline de ETL via **PySpark** na camada Silver para tratamento, limpeza de nulos, tipagem e modelagem distribuída de tabelas de alta volumetria.
* **Agregações Analíticas:** Utilização de **Spark SQL** na camada Gold para estruturação de visões consolidadas de negócio (Faturamento, performance logística e comportamento de categorias).
* **Entrega de Valor:** Integração dinâmica com o **Power BI**, aplicando conceitos rígidos de UI/UX para gerar um painel executivo focado na saúde da operação.

### 💵 2. Nexus Data Analytics: Pipeline em Tempo Real com PostgreSQL & PL/pgSQL
*Solução ponta a ponta voltada para automação de regras de negócio internas direto na camada de banco de dados relacionais.*
* **Automação no Banco:** Modelagem relacional com criação de **Triggers e Procedures em PL/pgSQL** para processar transações financeiras e volumetrias de forma reativa e imediata a cada `INSERT`.
* **Performance Otimizada:** Criação de **Views** agregadas para livrar o Power BI de queries custosas, acelerando o tempo de carregamento de relatórios.
* **Analytics Avançado:** Modelagem de indicadores financeiros em **DAX**, incluindo **Média Móvel de 7 dias** para identificar tendências estáveis e mitigar sazonalidades diárias.
